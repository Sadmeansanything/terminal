1. Посмотреть, где я - pwd
2. Создать папку - mkdir <foldername>
3. Зайти в папку - cd <foldername>
4. Создать три папки - mkdir folder_1 folder_2 folder_3
5. Зайти в любую папку - cd folder_1
6. Создать 5 файлов - touch test1.txt test2.txt test3.txt test4.json test5.json
7. Создать три папки - mkdir folder_1 folder_2 folder_3
8. Вывести список содерджимого папки - ls (или ls -la)
9. Открыть любой txt файл - vim test1.txt (откроет файл в текстовом редакторе vim) или cat test1.txt (просто выведет текст из файла)
10. Написать туда что-нибудь - i, <anytext>, <ESC>
11. Сохранить и выйти - :wq
12. Выйти из папки на уровень выше - cd ..
13. Переместить любые два файла - mv test1.txt folder_1/test1.txt test2.txt folder_2/test2.txt
14. cp test1.txt folder_1/test1.txt test2.txt folder_2/test2.txt
15. Найти файл по имени - find -name test1.txt
16. Создаем два файла, например fruits.txt и vegetables.txt (touch), далее с помощью текстового редактора добавляем поочередно в эти файлы любой текст с названиями ягод и овощей. Допустим, в обоих файлах у нас будет присутствовать название овоща potato. Результат команды grep -r potato выдаст, что в обоих файлах присутствует этот текст (название).
17. Вывести первые несколько строк из файла - head lesson1.txt
18. Вывести последние несколько строк из файла - tail lesson1.txt
19. Если файл слишком большой, можно воспользоваться командой less lesson1.txt, она выведет текст из файла постранично, с удобным просмотром. С помощью стрелок на клавиатуаре мы сможем перемещаться со строки на строку, с помощью PageUp и PageDown - со страницы на страницу. С помощью кнопки q - выйти из режима просмотра файла.
20. Вывести дату и время - date.
